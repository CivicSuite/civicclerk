CivicClerk

CivicClerk is planned municipal meeting software for agendas, packets, notices, minutes, motions, votes, action items, and public meeting archives.

Status: runtime foundation plus canonical schema, Alembic scaffolding, agenda item lifecycle enforcement, meeting lifecycle enforcement, packet snapshot versioning, notice compliance enforcement, immutable motion capture, immutable vote capture, action-item capture, and citation-gated minutes draft capture. IT staff can smoke-check the FastAPI app, inspect the schema plan, and exercise agenda, meeting, packet, notice, motion, vote, action item, and minutes citation rules, but archive and UI workflows are not implemented yet.

For clerks

Create draft meetings and agenda items through the API today; version packet snapshots, check notice compliance before public posting, capture immutable motions and votes, create action items linked to meeting outcomes, and create minutes drafts only when each sentence has citations and provenance.

For IT

Runtime foundation stack: FastAPI, civiccore==0.2.0, canonical schema metadata, Alembic migration scaffold, agenda lifecycle enforcement, meeting lifecycle enforcement, packet snapshot versioning, notice compliance enforcement, immutable motion/vote capture, action-item capture, citation-gated minutes drafts with provenance, pytest, docs verification, and placeholder-import CI gates. Planned local services include PostgreSQL 17, Redis 7.2, Celery, Ollama, and nginx-served React.

For residents

Planned public pages will clearly label posted agendas, packets, approved minutes, and archived meeting records.

What ships today

Documentation, governance files, issue templates, discussion seeds, a docs verification gate, a placeholder-import gate, Python package metadata, a FastAPI runtime foundation with root and health endpoints, canonical schema/Alembic scaffolding, agenda item lifecycle enforcement, meeting lifecycle enforcement, packet snapshot versioning, notice compliance enforcement, immutable motion/vote capture, action-item capture, and citation-gated minutes draft capture.
